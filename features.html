Key Features of Web Scanner

1. SQL Injection Detection

Identifies SQLi attempts in URL params and input fields using pattern-based filters and database error monitoring.

2. XSS Detection

Detects reflected, stored, and DOM-based XSS by inspecting scripts injected into DOM and query parameters.

3. DoS/DDoS Attack Simulation

Simulates flooding attacks to test how well the server handles abnormal traffic and alert thresholds.

4. Real-Time WebSocket Logging

Streams threat logs to frontend via WebSocket for live monitoring of security events and server logs.

5. JWT-Based Auth System

Secures APIs using JSON Web Tokens (JWT), ensuring only authorized users can scan or fetch logs.

6. Threat Log Export

Allows exporting scan logs in CSV format for documentation, audit, or compliance reports.

7. Admin Role Management

Admins can delete reports, manage users, and access full scanning analytics.

8. Rate Limiting

Applies Flask-Limiter to restrict excessive API usage, preventing abuse and spam.

9. Dark Mode Support

Toggle between light and dark themes to improve accessibility and user comfort.

10. Chart-Based Visualization

Presents threats in bar charts by severity, type, and frequency, making analysis easier.

11. Email Alert System

Sends instant email alerts for high-severity threats or suspicious activity detected.

12. User History Tracker

Keeps track of all scan reports submitted by a user for auditing and reference.

13. File Upload Monitoring

Scans file uploads for malware and other potential payloads before storing them.

14. URL Parameter Scanner

Examines all query strings for malicious injections or abnormal characters.

15. GeoIP Threat Mapping

Identifies and maps origin of threats based on IP location using GeoIP data.

16. Secure Session Handling

Implements secure cookies, token-based validation, and session expiry.

17. Two-Factor Authentication

Requires a second layer of authentication for secure user logins.

18. Captcha for Login

Prevents bot attacks by requiring image/text captcha at login & registration.

19. IP Blacklisting

Blocks repeat attackers or brute force attempts by blacklisting their IPs.

20. Severity Levels

Every vulnerability is tagged with a severity level: Low, Medium, or High.

21. Dynamic Payload Tester

Injects dynamic payloads to simulate hacker behavior in various fields.

22. AI-Based Pattern Learning

Learns common attack patterns using lightweight ML to improve detection.

23. API Abuse Detection

Tracks unusual API calls and flags behavior suggesting API misuse.

24. Security Headers Checker

Ensures HTTP headers like CSP, X-Frame-Options, and HSTS are enabled.

25. Live Scan Comparison

Compares real-time scan results with historical ones to detect changes.
